# Information

## Threads 

When a user opens a support thread in the main server, follow the steps below.

## Read the Thread

Before responding, read the whole post to understand what the user is asking for. Do not reply until you understand the issue fully.

If you understand the thread, respond to the issue directly in the post with consistently clear and helpful responses. Make sure to remain respectful at all times. 

If you do not understand the thread, respond to the user and try to troubleshoot the issue with them. If it's still unclear, ping a higher-up. Do not give in accurate information or guess anything. 

## Closing the Thread

Once you have resolved the issue, ask the user `Do you need any more help?`. If they are still in need of assistance, continue; if they don't, run the `;tl` command and then add the "Solved" tag to the post.

## Common Occurrences

Some users will request that you join their server to further help them with an issue. Keep in mind that you always have the right to refuse to join. If you feel comfortable, you can ask the user to either send you the invite via DMs or via the thread. Join their server and then continue the support. Note that you are never required to join an outside server other than our staff server.

If a user is ever creating troll posts, ping a moderator or an administrator.
